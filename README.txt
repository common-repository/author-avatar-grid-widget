Display a grid of user Gravatars based on criteria defined within widget settings

== Description ==

Need a pretty way to display all of your blog's authors? Want to display Gravatars for all of your site's subscribers?  The Author Avatar Grid Widget displays a grid of Gravatars for all users meeting the criteria defined within the widget settings. 

* Select to display all users or users of a specific role (administrator, editor, author, etcâ€¦)
* Choose to display only users with published posts
* Define the Gravatar size
